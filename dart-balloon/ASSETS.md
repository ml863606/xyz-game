# Assets

The first playable build uses procedural vector-style Godot drawing for all runtime visuals:

- Balloons: colored circles with highlights and string tails.
- Gold balloons: yellow variant with time reward feedback.
- Bombs: dark circular hazard with fuse.
- Dart: line and polygon drawing with colored tail fins.
- Background: sky, clouds, and fairground booth blocks.
- Effects: procedural burst particles and floating score text.

This keeps the first Douyin validation package simple and avoids external bitmap asset import risk.
