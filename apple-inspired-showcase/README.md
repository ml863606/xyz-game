# Aster Product Showcase

An original, Apple-inspired consumer technology homepage concept.

This project intentionally does not copy Apple Inc. trademarks, product names,
copy, or image assets. It mirrors the broad homepage rhythm of a premium device
site: slim navigation, full-width product announcements, a two-column promo
grid, a media strip, and a fine-print footer.

Open `index.html` directly in a browser.
